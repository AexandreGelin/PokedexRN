## Prérequis :
- Node.js (latest version)
- Expo
``` bash
  npm i -g expo-cli
```
- Un smartphone avec l'application Expo ou un navigateur web

## Demarrer le projet :

```bash
  npm install
```
```bash
  npm start
```

## API utilisée :

[Pokeapi](https://pokeapi.co/)
